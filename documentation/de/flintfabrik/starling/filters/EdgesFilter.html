Classpublic class EdgesFilter
InheritanceEdgesFilter starling.filters.FragmentFilter

The EdgesFilter applies a edge detection calculated with the Scharr-Operator (a variation of the Sobel Operator) combined with posterization. There are several options to achieve different effects, like "glowing edges", or comic like results by combining the edge detection with the posterized original.

See also
